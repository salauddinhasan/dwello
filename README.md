 # 🏠 Dwello - Room Booking Platform

A full-stack room booking application built with Next.js, TypeScript, MongoDB, and Tailwind CSS.

## 🌐 Live Demo

[https://dwello-nine.vercel.app](https://dwello-nine.vercel.app)

## ✨ Features

- 🏠 **Home Page** - 7 beautifully designed sections
- 🔍 **Explore Rooms** - Search, filter, sort & pagination
- 🏨 **Room Details** - Dynamic room details page
- 🔐 **Authentication** - Email/Password + Google OAuth
- ➕ **Add Rooms** - Protected route for hosts
- ✏️ **Edit/Delete** - Manage your own rooms
- 📱 **Fully Responsive** - Mobile, tablet & desktop
- 🎨 **Professional UI** - Airbnb-inspired design

## 🛠️ Tech Stack
